# CalTracker-Sho
AI powered calorie tracking app build in swift - Sho
## Setup Instructions

1. Clone this repository
2. Rename `Config.swift.example` to `Config.swift`
3. Open `CalTracker.xcodeproj` in Xcode
4. Build and run
